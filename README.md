# To-Do List Application 📋

โปรแกรมจัดการรายการงานแบบ CLI (Command Line Interface) ที่สามารถช่วยคุณจัดการงานในแต่ละวันได้อย่างมีประสิทธิภาพ

## ✨ คุณสมบัติ

- ✅ **เพิ่มงานใหม่** - บันทึกชื่องาน รายละเอียด และวันครบกำหนด
- ✅ **ดูรายการงาน** - แสดงรายการงานทั้งหมดในรูปแบบตาราง
- ✅ **แก้ไขงาน** - เปลี่ยนแปลงข้อมูลงาน หรือเปลี่ยนสถานะ
- ✅ **ลบงาน** - ลบงานที่ไม่ต้องการ พร้อมการยืนยัน
- ✅ **บันทึกข้อมูล** - เก็บข้อมูลงานลงในไฟล์ JSON อัตโนมัติ

## 📋 โครงสร้างข้อมูล

แต่ละงานจะมีข้อมูลดังนี้:
```json
{
  "id": 1,
  "title": "ชื่องาน",
  "description": "รายละเอียดของงาน",
  "due_date": "16/06/2026",
  "completed": false
}
```

## 🚀 การติดตั้ง

### ข้อกำหนด
- Python 3.6 ขึ้นไป
- ไลบรารี่ที่ใช้: `json`, `os` (มีมาตามค่าเริ่มต้นแล้ว)

### วิธีติดตั้ง

1. **Clone หรือดาวน์โหลดโปรเจค**
```bash
git clone https://github.com/sereezakung001-pixel/ToDaListAI.git
cd ToDaListAI
```

2. **ตรวจสอบเวอร์ชัน Python**
```bash
python --version
```

## 🎮 การรันโปรแกรม

เปิด Terminal/Command Prompt แล้วรันคำสั่ง:

```bash
python main.py
```

หรือ (ถ้าใช้ Python 3):
```bash
python3 main.py
```

## 📖 วิธีใช้งาน

เมื่อเริ่มโปรแกรม คุณจะเห็นเมนูหลักดังนี้:

```
==============================
     เมนู To-Do List
==============================
1. เพิ่มงานใหม่
2. ดูงานทั้งหมด
3. แก้ไขงาน
4. ลบงาน
5. ออกจากโปรแกรม
==============================
```

### ตัวอย่างการใช้งาน

#### 1️⃣ เพิ่มงานใหม่

```
เลือกตัวเลือก (1-5): 1

------------------------------
เพิ่มงานใหม่
------------------------------
ป้อนชื่อเรื่อง: ซื้อของเอา
ป้อนรายละเอียด: ไปห้างสรรพสินค้าซื้อน้ำ และขนม
ป้อนวันครบกำหนด (dd/mm/yyyy): 20/06/2026

✅ เพิ่มงาน 'ซื้อของเอา' สำเร็จแล้ว (ID: 1)
```

#### 2️⃣ ดูรายการงานทั้งหมด

```
เลือกตัวเลือก (1-5): 2

--------------------------------------------------------------------------------
รายการงานทั้งหมด
--------------------------------------------------------------------------------
ลำดับ  ชื่องาน                    วันครบกำหนด    สถานะ
--------------------------------------------------------------------------------
1      ซื้อของเอา                20/06/2026      ⏳ ยังไม่เสร็จ
2      ทำการบ้าน                22/06/2026      ✅ เสร็จแล้ว
--------------------------------------------------------------------------------
```

#### 3️⃣ แก้ไขงาน

```
เลือกตัวเลือก (1-5): 3

[แสดงรายการงาน]

------------------------------
แก้ไขงาน
------------------------------
เลือกลำดับงานที่ต้องการแก้ไข: 1

กำลังแก้ไข: ซื้อของเอา
1. แก้ไขชื่องาน
2. แก้ไขรายละเอียด
3. แก้ไขวันครบกำหนด
4. เปลี่ยนสถานะ (เสร็จแล้ว/ยังไม่เสร็จ)
0. ยกเลิก

เลือกตัวเลือก (0-4): 4
✅ เปลี่ยนสถานะเป็น 'เสร็จแล้ว' สำเร็จแล้ว
```

#### 4️⃣ ลบงาน

```
เลือกตัวเลือก (1-5): 4

[แสดงรายการงาน]

------------------------------
ลบงาน
------------------------------
เลือกลำดับงานที่ต้องการลบ: 1

กำลังจะลบ: ซื้อของเอา
รายละเอียด: ไปห้างสรรพสินค้าซื้อน้ำ และขนม

ต้องการลบงานนี้จริงหรือไม่ (y/n): y
✅ ลบงานสำเร็จแล้ว
```

#### 5️⃣ ออกจากโปรแกรม

```
เลือกตัวเลือก (1-5): 5
💾 บันทึกข้อมูลสำเร็จแล้ว
ออกจากโปรแกรม... ลาก่อน!
```

## 💾 การบันทึกข้อมูล

- ข้อมูลงานจะถูกบันทึกใน **ไฟล์ tasks.json** โดยอัตโนมัติ เมื่อคุณเลือกออกจากโปรแกรม
- เมื่อคุณเปิดโปรแกรมครั้งต่อไป งานเดิมของคุณจะถูกโหลดขึ้นมาโดยอัตโนมัติ

## 📁 โครงสร้างไฟล์

```
ToDaListAI/
├── main.py          # ไฟล์หลักของโปรแกรม
├── tasks.json       # ไฟล์เก็บข้อมูลงาน (สร้างจากการรันโปรแกรม)
└── README.md        # ไฟล์อธิบายโปรแกรม
```

## 🐛 ข้อมูลแก้ไขบัญชี

- ตรวจสอบให้แน่ใจว่าคุณป้อนลำดับงานที่ถูกต้อง
- วันครบกำหนดควรเป็นรูปแบบ `dd/mm/yyyy`
- หากไฟล์ tasks.json เสียหาย ให้ลบและรันโปรแกรมใหม่จะสร้างไฟล์ใหม่

## 📝 ตัวอย่างไฟล์ tasks.json

```json
[
  {
    "id": 1,
    "title": "ซื้อของเอา",
    "description": "ไปห้างสรรพสินค้าซื้อน้ำ และขนม",
    "due_date": "20/06/2026",
    "completed": true
  },
  {
    "id": 2,
    "title": "ทำการบ้าน",
    "description": "ทำใบงานคณิตศาสตร์",
    "due_date": "22/06/2026",
    "completed": false
  }
]
```
